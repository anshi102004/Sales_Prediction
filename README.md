# 📊 Sales Prediction using Machine Learning

## 📌 Project Overview

This project aims to predict product sales based on advertising expenditure using Data Science and Machine Learning techniques. The dataset contains information about advertising spending across different platforms such as **TV, Radio, and Newspaper**, and the goal is to analyze how these factors influence product sales.

The project includes **data preprocessing, exploratory data analysis (EDA), visualization, and model building** to understand the relationship between advertising budgets and sales.
